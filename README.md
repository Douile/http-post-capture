# http-post-listener

```
http-post-capture 0.1.0
A simple CLI program that listens on a specified address for HTTP post uploads and saves them in files

USAGE:
    http-post-capture [OPTIONS] -l <LISTEN_ADDRESS>

FLAGS:
    -h, --help       Prints help information
    -V, --version    Prints version information

OPTIONS:
    -l <LISTEN_ADDRESS>          The address to listen on
    -o <OUTPUT_DIRECTORY>        Directory to output to
```
